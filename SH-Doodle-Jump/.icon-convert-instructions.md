# Инструкции по конвертации SVG иконок в PNG

Эти SVG иконки созданы как placeholder для PWA приложения Doodle Jump. Их нужно конвертировать в PNG формат для полной совместимости с PWA manifest.

## Файлы в проекте

- `icon-192.svg` - иконка 192x192px
- `icon-512.svg` - иконка 512x512px

## Способ 1: Онлайн сервисы (Самый простой)

### CloudConvert
1. Перейдите на https://cloudconvert.com/svg-to-png
2. Загрузите `icon-192.svg` и `icon-512.svg`
3. В настройках установите:
   - Width: 192 (для icon-192) или 512 (для icon-512)
   - Height: 192 (для icon-192) или 512 (для icon-512)
4. Нажмите "Convert"
5. Скачайте получившиеся PNG файлы
6. Переименуйте в `icon-192.png` и `icon-512.png`

### Convertio
1. Перейдите на https://convertio.co/ru/svg-png/
2. Загрузите SVG файлы
3. Выберите формат PNG
4. Нажмите "Конвертировать"
5. Скачайте готовые PNG файлы

### Изображения для настройки (Online Image Editor)
1. Перейдите на https://www.iloveimg.com/ru/convert-to-jpg/svg-to-jpg
2. Загрузите SVG файлы
3. Выберите формат PNG вместо JPG
4. Скачайте результат

## Способ 2: Локальная конвертация через браузер

Создайте HTML файл `convert-svg-to-png.html` и откройте в браузере:

```html
<!DOCTYPE html>
<html>
<head>
    <title>SVG to PNG Converter</title>
</head>
<body>
    <h1>SVG to PNG Converter</h1>
    <input type="file" id="svgInput" accept=".svg" multiple>
    <div id="output"></div>

    <script>
        document.getElementById('svgInput').addEventListener('change', function(e) {
            const files = e.target.files;
            
            Array.from(files).forEach(file => {
                const reader = new FileReader();
                
                reader.onload = function(event) {
                    const svgData = event.target.result;
                    const img = new Image();
                    
                    img.onload = function() {
                        const canvas = document.createElement('canvas');
                        const size = file.name.includes('192') ? 192 : 512;
                        canvas.width = size;
                        canvas.height = size;
                        
                        const ctx = canvas.getContext('2d');
                        ctx.drawImage(img, 0, 0, size, size);
                        
                        canvas.toBlob(function(blob) {
                            const url = URL.createObjectURL(blob);
                            const a = document.createElement('a');
                            a.download = file.name.replace('.svg', '.png');
                            a.href = url;
                            a.textContent = `Download ${a.download}`;
                            a.style.display = 'block';
                            a.style.padding = '10px';
                            document.getElementById('output').appendChild(a);
                        }, 'image/png');
                    };
                    
                    img.src = 'data:image/svg+xml;base64,' + btoa(svgData);
                };
                
                reader.readAsText(file);
            });
        });
    </script>
</body>
</html>
```

Инструкция:
1. Сохраните код выше как `convert-svg-to-png.html`
2. Откройте файл в браузере
3. Выберите SVG файлы через input
4. Скачайте автоматически сгенерированные PNG файлы

## Способ 3: Использование командной строки (для продвинутых)

### Через Inkscape
```bash
inkscape icon-192.svg --export-type=png --export-filename=icon-192.png -w 192 -h 192
inkscape icon-512.svg --export-type=png --export-filename=icon-512.png -w 512 -h 512
```

### Через ImageMagick
```bash
convert -background none icon-192.svg -resize 192x192 icon-192.png
convert -background none icon-512.svg -resize 512x512 icon-512.png
```

### Через rsvg-convert
```bash
rsvg-convert -w 192 -h 192 icon-192.svg -o icon-192.png
rsvg-convert -w 512 -h 512 icon-512.svg -o icon-512.png
```

## После конвертации

1. Поместите получившиеся `icon-192.png` и `icon-512.png` в корень проекта
2. Проверьте, что в `manifest.json` указаны правильные пути к иконкам
3. Удалите строки с `.svg` иконками из manifest.json (они там как fallback)
4. Протестируйте PWA функционал

## Проверка PWA

После замены иконок на PNG:
1. Откройте DevTools в браузере (F12)
2. Перейдите на вкладку Application / Приложение
3. В разделе Manifest проверьте, что иконки отображаются корректно
4. Попробуйте установить PWA на устройство

## Альтернатива: Оставить SVG

Некоторые современные браузеры поддерживают SVG иконки в manifest.json. Если вы хотите использовать только SVG:
- Оставьте текущий manifest.json как есть
- Учтите, что старые браузеры могут не отображать иконки
- Рекомендуется всё же создать PNG версии для максимальной совместимости
