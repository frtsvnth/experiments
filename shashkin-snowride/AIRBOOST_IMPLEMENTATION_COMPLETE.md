## 🔧 **Как это работает:**

### **3-фазная система:**

#### **🚀 ФАЗА 1: Активный буст (20 кадров)**
```
- Сохраняется исходная позиция игрока
- Применяется импульс вперёд (airBoostVx = 2.0)
- Максимальное смещение ограничено 80px
- Создаются визуальные эффекты
```

#### **⏸️ ФАЗА 2: Завершение буста**
```
- Таймер буста заканчивается (duration = 0)
- Переход в фазу возврата (returnPhase = true)
- Вектор скорости сбрасывается
```

#### **↩️ ФАЗА 3: Возврат в исходную позицию**
```
- Плавное движение к исходной позиции
- Скорость возврата: 2 пикселя за кадр
- Автоматическое завершение при достижении цели
- Полный сброс системы
```
